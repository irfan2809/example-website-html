HTML Version of My Website
=========================

This folder contains a multi-page website using only HTML, CSS, and images. Each section is a separate HTML file, and navigation is done via links/buttons.

Pages:
- index.html: Home page with Groceries, Shopping, Transport buttons
- groceries.html: Groceries list with images and prices
- shopping.html: Shopping/electronics list with images and prices
- transport.html: Transport options (Bus, Train, Flight)
- bus.html: Bus booking details
- train.html: Train booking details
- flight.html: Flight booking details

To use:
- Open index.html in your browser and navigate using the buttons.
- All files are static and require no server or build tools.

You can further customize the content and styles by editing the HTML and CSS files.
